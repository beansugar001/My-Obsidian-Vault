# ARC-FSM

## 定义
Automatic security Rule Checking for Finite State Machine，在网表级别对 FSM 进行自动安全规则检查。

## 核心要点
1. Netlist-level 的 FSM 安全规则检查
2. SecFSM 的 Stage I 借鉴了其安全规则检查理念
3. 作为 SecFSM 的 benchmark 来源之一（ARC-FSM [25]）

## 代表工作
- [[SecFSM]]: 借鉴了规则检查理念，将检查从 netlist 层提前到 RTL 生成阶段
