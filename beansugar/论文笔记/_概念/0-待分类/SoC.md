# SoC

## 定义
System-on-Chip，将处理器、存储器、外设控制器等多种 IP 核集成在单一芯片上的系统。

## 核心要点
1. FSM 是 SoC 控制逻辑的核心组件
2. 多 IP 集成增加了攻击面
3. 硬件一旦流片无法修补，设计阶段安全至关重要

## 代表工作
- [[SecFSM]]: 面向 SoC 中安全关键 FSM 的 Verilog 生成
